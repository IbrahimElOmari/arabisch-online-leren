This documentation describes the design tokens used throughout the Arabisch Online Leren platform.
All colors are defined in HSL format and can be customized via CSS variables.

## Typography

### Font Sizes

| Token | Size | Usage |
|-------|------|-------|
| `--font-size-xs` | 0.75rem (12px) | Captions, labels |
| `--font-size-sm` | 0.875rem (14px) | Secondary text |
| `--font-size-base` | 1rem (16px) | Body text |
| `--font-size-lg` | 1.125rem (18px) | Emphasized text |
| `--font-size-xl` | 1.25rem (20px) | Subheadings |
| `--font-size-2xl` | 1.5rem (24px) | Section headings |
| `--font-size-3xl` | 1.875rem (30px) | Page headings |
| `--font-size-4xl` | 2.25rem (36px) | Hero headings |

## Spacing

The spacing scale follows Tailwind CSS defaults:

| Class | Value |
|-------|-------|
| `p-1` / `m-1` | 0.25rem (4px) |
| `p-2` / `m-2` | 0.5rem (8px) |
| `p-3` / `m-3` | 0.75rem (12px) |
| `p-4` / `m-4` | 1rem (16px) |
| `p-6` / `m-6` | 1.5rem (24px) |
| `p-8` / `m-8` | 2rem (32px) |

## Border Radius

| Token | Value | Usage |
|-------|-------|-------|
| `--radius` | 0.75rem | Default border radius |
| `rounded-sm` | 0.125rem | Small elements |
| `rounded-md` | 0.375rem | Medium elements |
| `rounded-lg` | 0.5rem | Large elements |
| `rounded-xl` | 0.75rem | Extra large elements |
| `rounded-full` | 9999px | Circular elements |


## Theme Variants

### Playful Theme (Under 16 years)
- Vibrant purple primary: `280 100% 70%`
- Golden accents
- Larger border radius: `1rem`
- Fun animations and glow effects

### Professional Theme (16+ years)
- Muted blue-gray primary: `220 25% 25%`
- Clean, subtle borders
- Smaller border radius: `0.5rem`
- Minimal, professional animations

## Usage in Components

### Using CSS Variables

```tsx
// In your component
<div className="bg-primary text-primary-foreground">
  Primary colored element
</div>

<div className="bg-destructive text-destructive-foreground">
  Destructive action
</div>

<div className="text-muted-foreground">
  Secondary text
</div>
```

### Using Tailwind Classes

```tsx
// Semantic colors
<Button variant="default">Primary Button</Button>
<Button variant="destructive">Delete</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="outline">Outline</Button>
```

## RTL Support

All components support Right-to-Left (RTL) layout for Arabic text:

```tsx
<div dir="rtl">
  <ArabicText>مرحبا بالعالم</ArabicText>
</div>
```

Use logical properties for spacing:
- `ms-4` instead of `ml-4` (margin-start)
- `me-4` instead of `mr-4` (margin-end)
- `ps-4` instead of `pl-4` (padding-start)
- `pe-4` instead of `pr-4` (padding-end)
